# 🚀 Gallery App Feature Roadmap

This document outlines potential future enhancements for the Gallery application to make it a world-class, professional-grade media manager.

## 🛠️ Small "Quality of Life" Polish (Quick Wins)
- **Power-User Keyboard Shortcuts:** Add global hotkeys. Press `i` to toggle Info, `t` for Tags, `f` for fullscreen, `space` to play/pause video, and `/` to instantly focus the search bar.
- **Custom Right-Click Context Menu:** When you right-click an image in the grid, instead of the ugly default browser menu, show a sleek glassmorphic menu with: *Copy Image, Reveal in File Explorer, Edit Tags, and Set as Wallpaper*.
- **Double-Click to Zoom:** In the Media Viewer, double-clicking the image should instantly fluid-zoom into the exact cursor location (and double-click again to zoom out).
- **Copy/Paste Tags:** A button to copy all tags from one image and quickly stamp them onto others.

## 🎨 Adaptive UI & Versatility (Medium Features)
- **Dynamic View Toggles:** Let the user switch between the current *Masonry View* (best for mixed aspect ratios), a *Square Grid View* (like Instagram, best for dense browsing), and a *Timeline View* (grouped automatically by Month/Year).
- **Resizable & Collapsible Sidebar:** Let the user drag the edge of the left sidebar to resize it, or collapse it completely to just show icons (maximizing screen real-estate for the gallery).
- **Video Hover Scrubbing:** In the main grid, hovering your mouse over a video thumbnail and moving left/right could "scrub" through the video preview silently without even opening it.
- **EXIF Metadata Reader:** For photos taken on cameras or phones, automatically extract and display the EXIF data in the Info panel (Camera Model, Lens, ISO, Shutter Speed, Aperture).

## 🚀 "Next-Gen" Modern Features (Big Additions)
- **Smart Search (AI Semantic Search):** Instead of just searching by filename or manual tags, we could integrate a lightweight local embedding model. This would let you type *"red car in the snow"* or *"receipt from starbucks"* and instantly find the image.
- **Drag-and-Drop Tagging / Organization:** Drag a tag from a master list and drop it onto images to tag them. Or drag an image onto a folder in the sidebar to physically move the file on your hard drive.
- **Lightweight Image Editor:** A built-in suite to Crop, Rotate, and adjust Brightness/Contrast/Saturation directly inside the app without needing to open Photoshop.
